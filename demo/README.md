# Demo

Bu dizin, sistemin uçtan uca çalışmasını gösteren demo senaryolarını içerir.

## Çalıştırma

```bash
# Proje kök dizininden
python demo/demo_scenario.py
```

## Demo İçeriği

Demo senaryosu aşağıdaki evrak türleri üzerinde sistemin tüm yeteneklerini gösterir:

1. **Dilekçe**: Bilgi işlem altyapı sorunu hakkında personel dilekçesi
2. **Üst Yazı**: Dijital Dönüşüm Eylem Planı hakkında tüm birimlere üst yazı
3. **Tutanak**: İnsan Kaynakları biriminin toplantı tutanağı

Her evrak için:
- ✅ OCR / Metin okuma
- ✅ Evrak türü sınıflandırma
- ✅ Anahtar bilgi çıkarma
- ✅ Eksik bilgi tespiti
- ✅ Mevzuat eşleştirme
- ✅ Özet oluşturma
- ✅ Resmi yazı taslağı
- ✅ Birim yönlendirme
- ✅ Kullanıcı bilgilendirme

## Jüri Sunumu İçin Not

- Demo gerçek zamanlı veya kayıttan sunulabilir
- Jürinin canlı çalıştırma talebine yanıt verilebilir
- İnternet kesintisine karşı yerel çalışma yedek planı mevcuttur
- Kullanılan veri setleri kurgu olup gerçek kamu verisi içermemektedir
